# 1236 성 지키기 

풀이 : 가로:N 세로:M 입력받음 

가로 세로를 2차원 배열 선언 후에 for문을 사용하여 입력을 받음 

boolean를 사용하여 가로 세로 배열 선언 

반복문을 반복하여 X를 탐색 탐색후에는 해당 배열에 트루 변경

거짓 개수 카운팅

가로와 세로 중에 카운팅이 더높을것을 값으로 출력

시행착오 : 난관 ? 

- 2차원 char배열을 BufferReader로 받으려고 했는데, \n이 들어가서, 잘 되지 않았다. 그래서 그냥 Scanner를 이용해서 했다. 

깨달은것

boolean은 자료형이다.
